## Key

- $RebufferRatio$ = $\frac{total\_rebuffer\_duration}{playback\_duration}$
*total_rebuffer_duration*: sum of (idle gaps > segment_duration) (from burst table)
*playback_duration*: last_burst_end - first_burst_start

- throughput (bps) = $\frac{{bytes\:delivered}}{time}$
- latency - time it takes for data to travel from one point to another in a network
